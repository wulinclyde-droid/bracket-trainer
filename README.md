# Bracket Queen Math Trainer

Bracket Queen is an innovative math training application designed to help users improve their arithmetic skills in a fun and engaging way. This trainer focuses on practicing and mastering various mathematical operations, offering users different modes to challenge their abilities and track their progress.

## Features
- **Interactive Interface:** Engaging and user-friendly design for all ages.
- **Multiple Difficulty Levels:** Choose from easy, medium, and hard to match your skill level.
- **Progress Tracking:** Monitor improvements over time and see detailed statistics.
- **Customizable Exercises:** Tailor sessions to focus on specific skills or types of math problems.

## Getting Started
To get started with using the Bracket Queen math trainer:
1. Clone the repository.
2. Install the necessary dependencies.
3. Run the application.
